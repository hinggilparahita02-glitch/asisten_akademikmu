## 1. Deskripsi Project
**Asisten Akademikmu** adalah aplikasi berbasis web yang dirancang untuk membantu taruna dalam mengelola aktivitas akademik harian secara terstruktur dan efisien. Aplikasi ini menyediakan fitur terpadu seperti dashboard akademik, kalender deadline, study timer, checklist rutinitas, catatan, pencapaian, prediksi IPK, serta simulasi target IPK.

---

## 2. Tujuan Project
- Membantu taruna mengatur tugas, deadline, dan target belajar
- Meningkatkan konsistensi belajar melalui study timer & streak
- Mempermudah pencatatan akademik (catatan & checklist)
- Memberikan motivasi belajar melalui sistem pencapaian
- Menyediakan simulasi prediksi IPK berbasis mata kuliah dan nilai
- Menjadi asisten akademik pribadi dalam satu aplikasi
